$$CV \quad for \quad skat \quad  team \quad cachalotes$$
Computer Vision system for detecting and recognition of armenian alphabet symbols. The system will be placed on Raspberry Pi 5, with camera C/CS IMX477R 12.3 Mp and lens CS2316ZM02 16mm. Base model: YOLOv8n.

the current version of the model is in runs\detect\bestmodel\weights\best.pt

the dataset with training, validation and test images can be found here: https://universe.roboflow.com/kiritonators-workspace/armenian_cv_2-mukco
